No. You log chat and save personal settings on your device. If streamers pay for us to host chat in the cloud, you can access that by clicking a button by your own choice.

No, these features work on the website in the browser.

By using this extension. A comment such as "Look at 3:21 into the video" will turn into a link to 3 minutes 21 seconds into the video upon loading.

How do I view live stream chat after a live stream is over?

By using this extension. Use this extension to log the chat, then you have several options afterwards. You can also use this extension menu to import a "Saved Chat" from someone else (or another platform) to load for this particular video. Finally, if a creator hosts the chats with us, you can load the chat in from the network and view it next to the past livestream.

Install/Support

RamblerTV is a web browser extension for desktop and Android for new, helpful features for using Rumble.com live streams.

It adds extra functionality to Rumble videos and livestreams which may be missing to you.

It can help you configure and search live stream chat to be more usable and useful to you.

Installation for desktop/laptop

  1. download the zip file and upzip it
  2. open your web browser and at chrome://extensions/ (example address)
  3. click "Load Unpacked" and select the folder you just unzipped
  4. you should see Rambler 1.0.0 active on this screen.
  5. questions? look up installing browser extensions on Chrome, the process is the same for all.

Installation for mobile/phone

  1. this has only been tested on Android phones currently.
  2. use a Chrome-based browser which can use browser extensions like the Kiwi browser or Samsung browser.
  3. download the zip file
  4. open your web browser and at kiwi://extensions/ (example address)
  5. click "load zip/crx" and select the download you just saved
  6. you should see Rambler 1.0.0 active on this screen.
  7. questions? look up installing browser extensions on Android, the process is the same for all.

Why isn't this on the Chrome Web Store?

We believe in the true open-source nature of this project and idea. Upon downloading, you or someone you trust can inspect the code for anything malicious. You can modify it to suit your needs. You or someone you know are probably smarter than I and can take this and improve upon it. This exists to be an open place of stream ideas in which people can add or subtract to and progress upon. With is as our motto, the best way seemingly to accomplish this is to provide it like this.

Don't like it?

If you do not like it, you can disable or uninstall it with no issues. All features can be turned on and off individually -- you choose which you want to use and how.

Usage and functionality

This provides functionality that can be used via the TAB and ENTER keys on the keyboard, a pop-up menu which can be used via touch or mouse, voice commands, and keyboard keybind commands. Each of these can be turned on or off as requested.

Usage

For this documentation, "livestream" will refer to the live video on https://rumble.com .

Basic Usage on Rumble LiveStreams:

Load an active livestream with chat on Rumble
if this is working, you will see loading messages pop up in the bottom right hand corner of the screen.
in the top right corner of your browser, you will see either a puzzle piece icon or a green keyboard icon. Click on it.
if puzzle piece, click on the pin icon next to the Rambler green keyboard icon.
if you see the green keyboard icon, click on it to toggle the extension Main Menu.
Main Menu
The menu with the icons the extension provides we will call the "Main Menu". This can be toggled in view or hidden.
On desktop, the Main Menu toggles on the right side of the chat.
On mobile, the Main Menu toggles below the video.
The main menu provides access to the options menu at its top, and different one-time chat toggle features below that.
From top to bottom, here are the icons in the Main Menu and their actions:
  • the cog activates the options menu
  • the crown is to toggle broadcaster only chat or all chat
  • the wrench is to toggle moderator only chat or all chat
  • the money bag is to toggle sub only chat or all chat
  • the money bag + wrench is to toggle sub and moderator only chat or all chat
  • the person in suit is to toggle verified/vip account only chat or all chat
  • the dollar sign is to toggle donation only chat or all chat
  • the @ sign is to toggle mention chat or all chat
  • the # sign is to toggle hashtag chat or all chat
  • the ab sign is to toggle text only chat or full chat (hide emojis, usernames, clutter)
  • the up arrow is to scroll to the top of the live chat
  • the down arrow is to scroll to the bottom of the live chat
  • the aA is to change the font size of chat
  • the left arrow is to toggle full width of video and chat sidebar
  • the up arrow is to toggle full height of video and chat sidebar
  • the magnifying glass brings up the Search Menu to toggle searching of the current stream chat
  • the magnifying glass + brings up the Search Menu to toggle searching of all saved stream chats on this device from Rumble
Search Menu
Search Menu: When the magnifying glass is clicked, the search menu appears.
  • When the search menu is active, the main menu icons work on the chat appearing in the search menu overlay
  • When the search menu is hidden, the main menu icons work on the chat appearing in the existing Rumble sidebar
  • When the search menu is active, type a phrase in the search bar.
  • The writing hand button will search saved chat for the phrase starting the message. Search for dog will find "dogs are cute" but NOT "your dog is cute."
  • The person button will search saved chat the user writing the message. Search for dog will find what the username dog wrote in the chat.
  • The notes on paper button will search saved chat for the phrase in the message. Search for dog will find "dogs are cute" AND "your dog is cute."
  • The calendar button will find all past stream events like hosts, hosts, gifts, clips and staff visits you have saved from this chat so far.
  • The full button will find all saved messages you have saved from this chat so far.
Options Menu
Options Menu: Press the cog icon in the main menu to open the options menu. The Options Menu is divided into these sections: turn on auto-saving of chat -- this turns on logging (to your device) of chat for reading back.
  • Theme
  • Highlight
  • Mute/Dim
  • Hide
  • Features
  • Saved Chats
  • Import Chats
  • Share
Saved Chats
This tab keeps the saved chats on your device for you to view at any time, export out or delete.
Import Chats
This tab you can import chat csv files to view on your device.
Share
This tab you can build your own share menu with your own share-to's for your device to use.

Advanced Usage

FEATURES EXPLAINED IN DEPTH

Basic Keybinds
By turning on this feature...
You can start using ALT+1, ALT+2, ALT+3, ALT+4 for controlling chat and screen commands on your desktop/laptop.
This is not recommended while gaming and streaming on same keyboard.
You can change these browser keybinds at chrome://extensions/shortcuts
but initially, here is what they are when you turn them on in this extension
Keybinds
Alt+1 = show/hide live stream chat
Alt+2 = show/hide recorded live stream chat
Alt+3 = search this live stream chat
Alt+4 = search all recorded live stream chat
Full Keybinds
By turning on this feature...
You can start using ALT+Q, ALT+W, etc... for controlling chat and screen commands on your desktop/laptop
This is not recommended while gaming and streaming on same keyboard.
Keybinds
ALT+Top Row Keyboard corresponds to the MAIN MENU left to right.
For example, the far left MAIN MENU icon toggles broadcaster messages,
and the far left keyboard key, ALT+Q, also toggles broadcaster messages.
And the keys continuing right (W,E,R...) map to moving right on the MAIN MENU.

 
Keybinds
ALT+Bottom Row Keyboard corresponds to the OPTIONS MENU.
ALT+Left Shift toggles the MAIN MENU
ALT+Z toggles the OPTIONS MENU
And the keys continuing right (X,C,V...) map to different sections of the OPTIONS MENU.
Voice Commands
By turning on this feature...
In the desktop/laptop Google Chrome version of the extension, you can test out using voice commands for controlling chat and screen commands
This is very experimental, use at your own risk.
This is not recommended while gaming and streaming on same keyboard.
This uses some computer memory, so if you are lagging, turn this off to save memory.
Commands
computer toggle chat
computer search chat
computer search all chat
computer chat top
computer chat bottom
Text to speech keywords
By turning on this feature...
Perhaps you are lurking or not actively reading every chat message, but would like to be alerted if someone in chat says a keyword.
Your desktop/laptop Google Chrome browser can speak a chat message out loud that contains a word(s) you specify.
This is very experimental, use at your own risk. Don't use very common words like hello/lol/etc.
This uses some computer memory, so if you are lagging, turn this off to save memory.
Auto-Saving Chat
By turning on this feature...
Chat is automatically logged/saved to this device (in the browser) as it displays on a live stream, so you can read it back later.
This uses some computer memory, so if you are lagging, turn this off to save memory.
Auto-Showing Chat
By turning on this feature...
Chat is automatically displayed on the video on a live stream instead of on the sidebar.
It turns your desktop/laptop into full screen mode where you see a full screen video and the chat flowing over top of it.
This uses some computer memory, so if you are lagging, turn this off to save memory.
Left Handed Chat
By turning on this feature...
Chat is on the left side of the video instead of the right side.
Upside Down Chat
By turning on this feature...
Chat is displayed newest at the top, oldest at the bottom. This is opposite of the current "flow down" chat.
Mouseover
By turning on this feature...
When you place your mouse cursor over the chat or emojis, they enlarge in size so that you can see them better.
Saved Chats
By turning on this feature...
While watching the current stream and chat, you can browse or search previous chats that you have saved or imported. Example: you can refer to previous conversations or continue on conversations and updates from day to day and stream to stream to build community.
You can export chats in CSV format so that you can use in personal spreadsheet or database.
Example: you can make custom analytics for yourself by exporting the chats out into a custom program where you can analyze how many raids you get per day, how many unique chatters per day, if/how many staff or verified users you had chatting, or whatever you can think of in a spreadsheet/database macro/formula/script.
Import Chats
By turning on this feature...
You can import chats to view or search from other Rumble streams or past streams on Twitch, Youtube, or Kick.
Share
By turning on this feature...
You can make your own share menu to use for your needs. For example, maybe you want a one click share your clips to your Discord server, your Minds account, your own website, or whatever you want. You can customize the share menu included to contain what you use.

VODS/Videos

This browser extension supplies a number of features for past streams on Rumble, also known as VODS.
For this documentation, "video" will refer to a VOD/Video On Demand/Previous Stream/Replay at https://Rumble.com/abc-something.html .
Some of these requested features are click on timestamps in comments, share links to specific parts of the video, and view chat on replay.

View previously saved chat on a video manually:

1a) watch a live stream and log the chat using this extension.
1b) or have someone give you the exported chat file they exported from a stream using this extension.
2a) load and watch the video the chat is from.
2b) Manually import the chat to view using this extension "Import Chat" in the options menu. 3) You can now watch the video replay while reading the chat from it.

Opening timestamped Rumble videos

Open a video such as example: https://Rumble.com/a1b2?t=120
The video should start playing 2 minutes in from the beginning according to that timestamp link. 120 seconds = 2 minutes
If you are using the extension, using ?t=seconds in a link will play the video that many seconds into the video.

Sharing timestamped Rumble video

Open a video such as example: https://Rumble.com/a1b2
Watch until some time in the video
If you are using the extension, open the options menu and click the SHARE category.
Click the "Share" button below to copy the general link to this video.
Click the "Share" button below again to copy the link to this specific time in the video.
Example: it will copy something like https://Rumble.com/a1b2?t=505 .
Paste this link somewhere else, or use the share menu to directly share it to a social platform.

Timestamp links on Rumble comments

Open a video with some comments such as example: https://Rumble.com/a1b2
Play the video.
If you see a comment below with a timestamp, such as "2:34 good point!", click on one.
The video should start playing fast-forwarding to the time in the video of the timestamp -- in this case at the 2:34 mark.

Scrolling, filtering, searching, etc...

On chat replays on videos, you are able to use most of the livestream functionality of Rambler to theme, scroll, filter, search, etc the chat on it to find particular chats or moments you have been looking for.
